
# Applica

Keep track of every job you apply for, monitor your progress through each hiring stage, and store important notes or details for every opportunity.

## Tech Stack

**Client:** Vue

**Server:** ASP.NET


## To Run the App

Run the command in bash

```bash
 chmod +x start-dev.sh
 ./start-dev.sh
```
